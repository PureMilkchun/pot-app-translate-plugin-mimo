# XiaoMi MiMo Translation Plugin for Pot

基于小米 MiMo-V2-Flash 大模型的 Pot 翻译插件。

## 功能特性

- ✅ 支持小米 MiMo-V2-Flash
- ✅ 支持自定义 API 地址
- ✅ 简单易用的配置界面
- ✅ 深色模式适配图标

## 安装使用

### 方法1：从 Release 下载（推荐）
1. 访问 [Releases 页面](https://github.com/pot-app/pot-app-translate-plugin-mimo/releases)
2. 下载最新的 `plugin.com.pot-app.mimo.potext` 文件
3. 在 Pot 应用中导入插件



## 配置说明

### 必需配置

- **API密钥**: 您的小米 MiMo API 密钥

### 可选配置

- **模型选择**: 
  - `mimo-v2-flash` (默认) - 快速轻量模型
- **API地址**: 自定义 API 地址（默认使用 `https://api.xiaomimimo.com/v1`）

## 支持语言

支持所有 Pot 常见语言：
- 自动检测 (auto)
- 中文 (zh_cn, zh_tw)
- 英语 (en)
- 日语 (ja)
- 韩语 (ko)
- 法语 (fr)
- 西班牙语 (es)
- 俄语 (ru)
- 德语 (de)
- 意大利语 (it)
- 土耳其语 (tr)
- 葡萄牙语 (pt_pt, pt_br)
- 越南语 (vi)
- 印度尼西亚语 (id)
- 泰语 (th)
- 马来语 (ms)
- 阿拉伯语 (ar)
- 印地语 (hi)
- 蒙古语 (mn_cy, mn_mo)
- 高棉语 (km)
- 挪威语 (nb_no, nn_no)
- 波斯语 (fa)

## 获取 API 密钥

1. 访问小米开发者平台：https://mimo.xiaomi.com
2. 注册开发者账号
3. 申请 MiMo API 访问权限
4. 获取 API 密钥
